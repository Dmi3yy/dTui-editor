# Glossary

This glossary defines stable terms used across dTui Editor documentation.

| Term | Meaning |
| --- | --- |
| dTui Editor | The Evolution CMS package that integrates TOAST UI Editor. |
| TOAST UI Editor | The upstream browser editor used by this package. |
| evo-ui | The shared Evolution manager UI toolkit and field lifecycle layer. |
| Rich editor | An Evolution manager textarea enhanced through rich editor events. |
| Profile | A dTui configuration group for toolbar, options, and plugins. |
| EVO link | An internal Evolution resource link inserted through the `E` button. |
| Placeholder link | Evolution link format such as `[~12~]`. |
| Paste upload | Clipboard image upload handled by `dtui-image-upload`. |
| UML block | A PlantUML custom block rendered by the UML plugin. |
| Prism bundle | The local syntax highlighting language bundle. |
| Asset bridge | The boundary where evo-ui initializes a field and dTui owns editor assets. |
