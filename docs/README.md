# dTui Editor Documentation

dTui Editor is a TOAST UI Editor integration for Evolution CMS and a rich text
editor package for manager fields.

Choose a language:

- [English](en/README.md)
- [Ukrainian](uk/README.md)
- [Polish](pl/README.md)
- [German](de/README.md)
- [French](fr/README.md)

The documentation covers manager usage, image workflows, EVO links, UML diagrams,
code highlighting, configuration, routes, assets, and theme behavior.
